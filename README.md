# Object Detection Tool (Scratch)

This is a simple **Object Detection Tool** built using **Scratch** and **Google Teachable Machine**.  
It detects objects in real time using a camera and shows the result instantly.

## Tools Used
- Scratch 3.0  
- Google Teachable Machine  
- Webcam
- Machine Learning (Image Classification)

## How It Works
1. Train the model in Teachable Machine  
2. Connect the model to Scratch  
3. The camera detects objects and displays the output  

## How to Run
1. Open the `.sb3` file in Scratch  
2. Allow camera access  
3. Click the green flag
